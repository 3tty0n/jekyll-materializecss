# jekyll-materializecss
a simple theme for [Jekyll](https://github.com/jekyll/jekyll) using [Materializecss](https://github.com/Dogfalo/materialize) 

## Using
   - Clone this repository  
   - ``` bundle install ```  
   - ``` jekyll serve --watch ```
